Bootstrap 3

In addition to all of our custom classes, all of the Bootstrap 3 classes and features can be used with BootSassy

Responsive Layout

BootSassy is 100% responsive and looks great on all devices and screen sizes. We use the Bootstrap grid classes

JavaScript Widgets

BootSassy comes packaged with all Bootstrap JavaScript widgets as well as a portfolio gallery, image scroller and lightboxes

SASS

Be sure to use the SASS files (.scss) when creating your template. SASS offers the following features...

  • CSS Variables
  • Nested Elements
  • Importing / Partials
  • Conditional Statements
  • Mixins
  • Color Functions
